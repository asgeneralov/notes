# Java Messaing Service

Это API для работы с MOM (Message-Oriented Middleware)

## Ключевые характеристики MOM:

- Asynchronous Communication – асинхронная коммуникация между приложениями
- Decoupling – слабая связанность между приложениями
- Reliability – надежность доставкит (опционаольно)
- Scalability – масштабируемость (опционаольно)

JMS поддерживает два наиболее распространенных стиля обмена сообщениями:
- Point To Point (PTP) - позволяет клиенту отправить сообщение в абстракцию, называемую – очередь, а другому клиенту прочитать сообщение из этой конкретной именованной очереди.
- Publish Subscribe (Pub/Sub) - позволяет клиенту отправить сообщение множеству других клиентов одновременно используя абстракцию, называемую – топик. Множество клиентов может подписаться на интерисующий их топик и получать сообщения.
